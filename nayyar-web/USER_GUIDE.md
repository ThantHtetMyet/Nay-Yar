# Nay-Yar Web Portal အသုံးပြုသူလက်စွဲ

ဤလက်စွဲစာအုပ်သည် Nay-Yar Web Portal ကိုအသုံးပြုပြီး အိမ်ခြံမြေရှာဖွေခြင်း၊ အကောင့်စီမံခန့်ခွဲခြင်းနှင့် ကြော်ငြာတင်ခြင်းများပြုလုပ်ပုံကို ရှင်းလင်းဖော်ပြထားပါသည်။

> **မှတ်ချက်:** လုပ်ဆောင်ပုံအဆင့်ဆင့်ကို ပိုမိုရှင်းလင်းစေရန် သက်ဆိုင်ရာကဏ္ဍများတွင် ဓာတ်ပုံ (Screenshots) များ ထည့်သွင်းသင့်ပါသည်။

## ၁။ စတင်အသုံးပြုခြင်း

### Portal သို့ဝင်ရောက်ခြင်း
Web browser ကိုဖွင့်ပြီး application URL သို့သွားပါ။ ဖုန်းရော ကွန်ပျူတာပါ အဆင်ပြေစွာ အသုံးပြုနိုင်ပါသည်။

### Browser ခွင့်ပြုချက်များ (Browser Permissions)
အကောင်းဆုံး အသုံးပြုမှုအတွေ့အကြုံရရှိရန် browser ၏ location access (တည်နေရာအသုံးပြုခွင့်) ကို 'Allow' ပေးထားပါ။ သို့မှသာ သင့်အနီးနားရှိ အိမ်ခြံမြေများကို "Locate Me" စနစ်ဖြင့် ရှာဖွေနိုင်မည်ဖြစ်သည်။

## ၂။ ပင်မစာမျက်နှာနှင့် မြေပုံအသုံးပြုခြင်း

ပင်မစာမျက်နှာ (Home Page) တွင် အိမ်ခြံမြေနေရာများကို ပြသထားသည့် မြေပုံပါဝင်ပါသည်။

![Screenshot of Home Page Map]

### မြေပုံကိုအသုံးပြုခြင်း
- **ရွှေ့ခြင်း (Pan):** မြေပုံကို ရွှေ့ရန် click နှိပ်ပြီး ဆွဲပါ (သို့မဟုတ် ဖုန်း screen ပေါ်တွင် လက်ဖြင့်ဆွဲပါ)။
- **ချဲ့ခြင်း/ချုံ့ခြင်း (Zoom):** `+` နှင့် `-` ခလုတ်များ သို့မဟုတ် mouse scroll ကိုသုံးပါ။
- **အမှတ်အသားများ (Markers):** အိမ်ခြံမြေအမှတ်အသားများကို နှိပ်ပြီး အချက်အလက်အကျဉ်းချုပ်ကို ကြည့်ရှုနိုင်ပါသည်။

### "Locate Me" (လက်ရှိနေရာရှာဖွေခြင်း) စနစ်
သင့်လက်ရှိရောက်နေသောနေရာကို မြေပုံပေါ်တွင် ဗဟိုပြုရန် **Locate Me** ခလုတ် (ပစ်မှတ်ပုံစံ) ကိုနှိပ်ပါ။
- အကယ်၍ ခွင့်ပြုချက်တောင်းခံလာပါက **Allow** ကိုနှိပ်ပါ။
- အကယ်၍ location access ကို ပိတ် (Block) ထားမိပါက browser settings တွင် ပြန်လည်ပြင်ဆင်ရန် ညွှန်ကြားချက် (Error message) ပေါ်လာပါလိမ့်မည်။

![Screenshot of Location Permission Prompt]

## ၃။ အကောင့်ဝင်ရောက်ခြင်းနှင့် စာရင်းသွင်းခြင်း

ကြော်ငြာတင်ရန် သို့မဟုတ် အချက်အလက်များသိမ်းဆည်းရန် အကောင့်ဝင်ထားရန် လိုအပ်ပါသည်။

### အကောင့်အသစ်ဖွင့်ခြင်း (Sign Up)
1. Navigation bar ရှိ **Sign Up** ခလုတ်ကို နှိပ်ပါ။
2. အီးမေးလ်၊ အသုံးပြုသူအမည် (Username) နှင့် စကားဝှက် (Password) တို့ကို ဖြည့်သွင်းပါ။
3. **Register** ကိုနှိပ်ပြီး အကောင့်ဖွင့်ပါ။

### အကောင့်ဝင်ခြင်း (Login)
1. **Sign In** ခလုတ်ကို နှိပ်ပါ။
2. စာရင်းသွင်းထားသော အီးမေးလ်နှင့် စကားဝှက်ကို ရိုက်ထည့်ပါ။
3. **Login** ကိုနှိပ်ပြီး ဝင်ရောက်ပါ။

![Screenshot of Login/Signup Modal]

## ၄။ မိမိအကောင့်ကို စီမံခန့်ခွဲခြင်း

အကောင့်ဝင်ပြီးနောက် **User Profile** ပုံကိုနှိပ်ပြီး "My Account" သို့ ဝင်ရောက်ပါ။

### အကောင့်အချက်အလက်များ (Account Details)
- **Edit Profile:** အသုံးပြုသူအမည်၊ အီးမေးလ် သို့မဟုတ် ဖုန်းနံပါတ်ကို ပြင်ဆင်နိုင်သည်။
- **Change Password:** လုံခြုံရေးအတွက် စကားဝှက်ကို ပြောင်းလဲနိုင်သည်။
  - စကားဝှက်အဟောင်းကို ရိုက်ထည့်ပါ။
  - စကားဝှက်အသစ်ကို နှစ်ကြိမ်ရိုက်ထည့်ပြီး အတည်ပြုပါ။

![Screenshot of My Account Modal]

### မိမိတင်ထားသော အိမ်ခြံမြေများ (My Properties)
**Properties List** tab တွင် မိမိတင်ထားသော ကြော်ငြာအားလုံးကို ကြည့်ရှုနိုင်ပါသည်။
- အိမ်ခြံမြေတစ်ခုချင်းစီကို နှိပ်ပြီး အသေးစိတ်ကြည့်ရှုနိုင်သည်။
- ကြော်ငြာကို ပြင်ဆင်လိုပါက **Edit** သို့မဟုတ် ဖျက်လိုပါက **Delete** ကိုနှိပ်ပါ။

## ၅။ အိမ်ခြံမြေကြော်ငြာတင်ခြင်းနှင့် စီမံခန့်ခွဲခြင်း

### ကြော်ငြာအသစ်တင်ခြင်း
1. Navigation bar ရှိ **Create Post** ခလုတ်ကို နှိပ်ပါ။
2. **အိမ်ခြံမြေ အချက်အလက်များ** ဖြည့်သွင်းပါ:
   - ခေါင်းစဉ်၊ အကြောင်းအရာ၊ ဈေးနှုန်း၊ အကျယ်အဝန်း၊ အိပ်ခန်းအရေအတွက် စသည်တို့။
   - အိမ်ခြံမြေအမျိုးအစား (ကွန်ဒို၊ လုံးချင်း၊ တိုက်ခန်း စသည်) ရွေးချယ်ပါ။
3. **ဓာတ်ပုံများ (Upload Images):** ဓာတ်ပုံများကို ဆွဲထည့်ပါ (Drag and drop) သို့မဟုတ် ရွေးချယ်တင်ပါ။
4. **တည်နေရာ (Location):** လိပ်စာ သို့မဟုတ် စာတိုက်ကုဒ် (Postal code) ပေါ်မူတည်ပြီး မြေပုံပေါ်တွင် နေရာအလိုအလျောက်ပြပါမည်။ လိုအပ်ပါက pin ကို ကိုယ်တိုင်ရွှေ့ပြောင်းနိုင်သည်။
5. **Submit** ကိုနှိပ်ပြီး ကြော်ငြာတင်ပါ။

![Screenshot of Create Post Form]

### ကြော်ငြာကို ပြင်ဆင်ခြင်း
1. **My Account** > **Properties List** သို့သွားပါ။
2. ပြင်ဆင်လိုသော ကြော်ငြာကို ရွေးချယ်ပါ။
3. အချက်အလက်များ သို့မဟုတ် ဓာတ်ပုံများကို ပြင်ဆင်ပါ။
4. **Save Changes** ကိုနှိပ်ပြီး သိမ်းဆည်းပါ။

## ၆။ အခက်အခဲများကို ဖြေရှင်းခြင်း

### တည်နေရာအသုံးပြုခွင့် ငြင်းပယ်ခံရပါက (Location Access Denied)
"Location Access Denied" ဟုပေါ်လာပါက:
1. Browser address bar ရှိ **Lock** (🔒) သို့မဟုတ် **Info** (ⓘ) icon ကိုနှိပ်ပါ။
2. **Location** setting ကိုရှာပြီး **Ask (default)** သို့မဟုတ် **Allow** သို့ပြောင်းပေးပါ။
3. စာမျက်နှာကို refresh လုပ်ပြီး ပြန်လည်ကြိုးစားပါ။

![Screenshot of Browser Location Settings]

### အကောင့်ဝင်မရဖြစ်ပါက (Login Issues)
- အီးမေးလ်မှန်ကန်မှုရှိမရှိ စစ်ဆေးပါ။
- စကားဝှက်မေ့သွားပါက login စာမျက်နှာရှိ **Forgot Password** link မှတဆင့် စကားဝှက်ကို ပြန်လည်ရယူပါ။
